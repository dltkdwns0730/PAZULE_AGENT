# v3.3.6 Login Height Scroll Fix

## Purpose

`/login` 화면에서 브라우저 세로 크기를 조정할 때 로그인 콘텐츠 일부가 보이지 않는 문제를 수정한다.

## Root Cause

공통 `Layout`의 phone frame은 고정된 `100dvh` 높이와 `overflow-hidden`을 사용한다. 그 안에서 `Login` 화면도 다시 `min-h-dvh`를 사용해 부모 frame보다 큰 자식 높이를 만들 수 있었고, 로그인 화면 자체에는 스크롤 책임이 없어 세로 리사이즈 시 내용 접근이 막혔다.

## Changes

- `Login` 루트 컨테이너를 부모 frame 기준의 `h-full min-h-0 overflow-y-auto` 레이아웃으로 변경했다.
- 로그인 본문은 `min-h-full`을 사용해 기존 중앙 정렬 의도를 유지하면서, 콘텐츠가 frame보다 커질 경우 내부 스크롤로 접근 가능하게 했다.
- 로그인 주요 액션 렌더링과 스크롤 레이아웃 클래스를 고정하는 회귀 테스트를 추가했다.
