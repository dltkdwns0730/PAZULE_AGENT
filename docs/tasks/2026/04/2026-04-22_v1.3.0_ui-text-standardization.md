# 📋 작업 명세서: 미션 UI 텍스트 정문화

## 1. 배경 및 목적
- **배경:** 미션 제출 화면 및 홈 화면에서 특정 정답에 의존적인 하드코딩된 텍스트("거대한 책장을 찾아라")가 노출됨.
- **목적:** 모든 미션에 범용적으로 적용 가능한 용어로 UI 텍스트를 변경하여 사용자 경험의 일관성을 확보.

- [x] `front/src/pages/PhotoSubmission.jsx`:

    - 분위기 미션 제목 변경: "분위기를 포착하라" -> "분위기 담아내기"

## 3. 검증 결과 및 품질
- **확인:** 프론트엔드 소스 코드 내 하드코딩된 문자열 검색 결과, 변경 대상 텍스트가 모두 수정되었음을 확인.
- **일관성:** 홈 화면과 제출 화면의 명칭을 통일하여 사용자 혼선을 방지함.

- **특이 사항:** 주석 및 내부 변수명은 변경하지 않았으나, 사용자에게 노출되는 레이블은 모두 수정됨.
