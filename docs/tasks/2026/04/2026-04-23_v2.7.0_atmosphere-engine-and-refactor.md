# 📋 작업 명세서: 분위기 엔진 v2.7 및 스크립트 구조 최적화

## 1. 배경 및 목적
- **배경:** 기존 분위기 판정이 단순 키워드 매칭에 의존하여 변별력이 낮고, `scripts/` 디렉토리에 도구가 난립하여 유지보수가 어려움.
- **목적:** 시각적으로 대비되는 5개 표준 클러스터를 도입하여 판정 정확도를 높이고, 프로젝트 구조를 체계화함.

## 2. 구현 세부 사항

### 2.1 분위기 엔진 (Atmosphere Engine v2.7)
- [x] **표준 클러스터 정의:** Vibrant, Serene, Vintage, Majestic, Whimsical 5종 정의.
- [x] **Ground Truth 구축:** `data/atmosphere_ground_truth_siglip2.json` 생성. (에셋별 정답 멀티라벨링 완료)
- [x] **스마트 피드백 (`app/core/hints.py`):** Target vs Actual 점수 비교를 통한 맞춤형 힌트 로직 구현.
- [x] **이미지 필터링:** 단색 이미지(STD < 5.0) 필터링으로 VLM 환각 방지 로직 추가.

### 2.2 프로젝트 리팩토링
- [x] **스크립트 재배치:** `scripts/` 루트의 파일을 `tools/`, `debug/`, `benchmarks/`로 이동.
- [x] **레거시 제거:** SigLIP2로 대체된 `app/models/clip.py` 제거 및 관련 참조 수정.
- [x] **문서 업데이트:** `architecture.md` 및 `refactor-audit.md`에 v2.7.0 변경 사항 반영.

## 3. 검증 결과 및 품질
- **변별력 향상:** 5개 클러스터 기반 SigLIP2 판정 시 정답과 오답 간의 스코어 갭이 0.2 이상 확보됨을 확인.
- **구조적 개선:** 스크립트 카테고리화로 목적별 도구 접근성 향상 및 코드 베이스 슬림화.

## 4. 핸드오버 참고 사항
- 분위기 미션의 `PASS_THRESHOLD`는 현재 0.35를 기준으로 권장함.
- 향후 새로운 에셋 추가 시 `data/atmosphere_ground_truth_siglip2.json`에 해당 분위기 GT를 반드시 추가해야 함.
