# v3.2.1 Admin Console Korean Localization

## Background
The admin login and operations console still showed English labels, empty-state
copy, and action text after the multicompany admin scope work.

## Goal
Localize `/admin/login` and `/admin` visible admin UI text into Korean while
preserving the existing layout and operational console style.

## Proposed Change
- Translate admin login title, helper copy, Google login button, and fallback
  error text.
- Translate admin sidebar/mobile navigation and logout label.
- Translate dashboard KPI labels, recent activity title, empty state, and
  company filter label.
- Translate admin logs, coupons, users, and mission detail labels for consistent
  navigation from `/admin`.

## Validation Plan
- `npm run build`
- `npm run lint`
- Confirm `http://127.0.0.1:5173/admin/login` responds locally.

## Actual Results
- Frontend production build passed with the existing large chunk warning.
- Frontend lint passed with existing unrelated hook dependency warnings.
- Local dev server responded on `/admin/login`.

## Rollback
Revert the frontend copy changes in admin pages/components/services to restore
the prior English admin UI.
